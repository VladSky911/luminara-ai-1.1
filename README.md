# Luminara AI 1.1

**Прозрачная RAG База Знаний** — AI-приложение для семантического поиска и чата с документами, с полным цитированием источников, отслеживанием поиска и оценкой уверенности.

## 🎯 О проекте

Luminara AI 1.1 — это интеллектуальная база знаний на основе RAG (Retrieval-Augmented Generation), которая позволяет:

- 📄 **Загружать документы** (PDF, DOCX, TXT, MD) и автоматически разбивать их на чанки
- 🔍 **Семантический поиск** по базе знаний с векторными эмбеддингами
- 💬 **Чат с AI** — ответы основаны на ваших документах с цитатами и источниками
- 📊 **Мониторинг системы** — здоровье БД, статистика, производительность
- ⚙️ **Настройки** — размер чанков, модель, параметры поиска

### Ключевые особенности

| Возможность | Описание |
|-------------|----------|
| 🔗 Цитирование источников | Каждый ответ AI содержит ссылки на конкретные фрагменты документов |
| 🎯 Оценка уверенности | AI показывает уровень уверенности в ответе |
| 🔍 Режимы поиска | Strict / Balanced / Creative — контроль точности vs креативности |
| 📑 Сравнение документов | Параллельное сравнение чанков из разных документов |
| 📋 Шаблоны чата | Предустановленные шаблоны для типовых запросов |
| 🔔 Уведомления | Real-time уведомления о событиях в системе |
| ⌨️ Горячие клавиши | Command Palette (Ctrl+K) для быстрой навигации |
| 🌙 Тёмная / Светлая тема | Переключение между темами |
| 📱 Адаптивный дизайн | Поддержка мобильных устройств |

## 🚀 Быстрый старт

### Предварительные требования

- **Node.js** 18+ ([скачать](https://nodejs.org/))
- **npm** или **bun**

### Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/VladSky911/luminara-ai-1.1.git
cd luminara-ai-1.1

# 2. Установить зависимости
npm install

# 3. Создать файл окружения
copy .env.example .env        # Windows
# cp .env.example .env        # macOS/Linux

# 4. Инициализировать базу данных
npx prisma db push

# 5. Запустить dev-сервер
npm run dev
```

Открыть **http://localhost:3000** — готово! 🎉

## 🖥 Страницы приложения

### 📄 Documents (Документы)
Загрузка, просмотр и управление документами. Поддержка PDF, DOCX, TXT, MD. Автоматическая разбивка на чанки.

### 🔍 Search (Поиск)
Семантический поиск по базе знаний с фильтрами по типу документа, статусу и тегам. Векторный поиск с оценкой релевантности.

### 💬 Chat (Чат)
Интерактивный чат с AI. Три режима: Strict (только факты), Balanced (баланс), Creative (креативный). Цитирование источников в каждом ответе.

### 📊 Dashboard (Панель)
Визуализация статистики: количество документов, чанков, сессий чата. Графики и метрики производительности.

### 🔔 Notifications (Уведомления)
Real-time уведомления о событиях: новые документы, завершение индексации, ошибки.

### ⚙️ Settings (Настройки)
Конфигурация системы: размер чанков, модель, параметры поиска, тема оформления.
